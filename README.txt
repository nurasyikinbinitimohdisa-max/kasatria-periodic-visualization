# Kasatria 3D Visualization Assignment

This project demonstrates a web-based 3D data visualization built using Three.js.

## Features
- Google Sign-In (basic authentication gate)
- Data fetched dynamically from Google Sheets (CSV format)
- 3D grid visualization using Three.js
- Color encoding based on Net Worth
- Profile images rendered from sheet data
- Interactive camera controls (rotate, zoom, pan)

## Tech Stack
- HTML, JavaScript
- Three.js (ES Modules)
- Google Sheets (CSV Publish)
- Live Server (local development)

## How to Run
1. Open the project folder in VS Code
2. Right click `index.html` or `periodic.html`
3. Select **Open with Live Server**
4. The visualization will load in the browser

> Recommended browser: Google Chrome

## Notes
- Internet connection is required to load Google Sheet data and images
- ES Modules are used as recommended by the latest Three.js documentation
